# Vonage Client SDK Video iOS

The Vonage Client SDK Video iOS lets you use Vonage-powered video sessions in apps you build for iPad, iPhone, and Mac devices.

For full documentation, installation instructions, system requirements, and guides, visit the **[Vonage Video iOS SDK overview](https://developer.vonage.com/en/video/client-sdks/ios/overview)** on the Vonage API Developer Portal.

## Installation

Add the SDK to your project using Swift Package Manager:

```
https://github.com/vonage/vonage-video-client-sdk-swift.git
```

In Xcode, go to **File > Add Package Dependencies…**, paste the URL above, and select the **VonageClientSDKVideo** library.

> For full installation instructions (including CocoaPods and migration from CocoaPods), see the [Installation section](https://developer.vonage.com/en/video/client-sdks/ios/overview#installation) of the developer portal.
